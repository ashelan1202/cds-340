5 Things I've Learned Since the Course Started

  • Main thing I've learned: Use proper editors for HTML files to prevent most errors.

  • CSS can be written within an HTMl file, you don't always have to keep them separate.
  • Colors in HTML and CSS can be represented in 3 ways: by name, by hex code, or by the RGB values of the color.
